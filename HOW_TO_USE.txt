🚀 শুরু করো (3 সেকেন্ডে):
═══════════════════════════════════════════════════════════════

1. "start_chatrag.bat" এ double-click করো
2. Browser এ UI খুলবে
3. শুরু করো!


📤 Documents Upload করো:
═══════════════════════════════════════════════════════════════

1. UI তে "Choose Files" button এ click করো
2. তোমার files select করো (PDF, TXT, CSV)
3. Files automatically upload হবে
4. "Process Documents" button এ click করো
5. Wait করো - processing complete হলে "Ready" দেখাবে


💬 Chat করো:
═══════════════════════════════════════════════════════════════

1. নিচের message box এ তোমার প্রশ্ন লেখো
2. Enter press করো
3. AI উত্তর দেবে তোমার documents থেকে!


📝 Example Questions:
═══════════════════════════════════════════════════════════════

🎯 Features:
═══════════════════════════════════════════════════════════════

✅ Multiple files upload করতে পারবে
✅ PDF, TXT, CSV support করে
✅ Real-time chat
✅ Source documents দেখাবে
✅ Beautiful UI
✅ Fast & accurate answers


🔧 Troubleshooting:
═══════════════════════════════════════════════════════════════

❌ "Offline" দেখাচ্ছে?
   → API server চালু আছে কিনা check করো
   → start_chatrag.bat আবার run করো

❌ "Upload docs first" দেখাচ্ছে?
   → Documents upload করো
   → "Process Documents" button click করো

❌ Answer পাচ্ছো না?
   → Documents properly process হয়েছে কিনা check করো
   → Status "Ready" দেখাচ্ছে কিনা check করো


📞 Quick Commands:
═══════════════════════════════════════════════════════════════

Test API:             python test_api.py

💡 Tips:
═══════════════════════════════════════════════════════════════

• একসাথে multiple files upload করতে পারো
